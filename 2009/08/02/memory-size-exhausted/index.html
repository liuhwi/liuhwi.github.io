PHP内存溢出解决方法

发表于 2009-08-02 | | 阅读次数

PHP报错此种错误

Allowed memory size of xxx bytes exhausted at xxx:xxx (tried to allocate xxx bytes)

出现该错误的原因

是因为PHP页面消耗的最大内存小，如果文件太大 或图片太大 在读取的时候 会发生上述错误。

解决办法

1. 修改 php.ini

将memory_limit由 64M 改成 128M（或更大），重启PHP服务（apache+php，重启apache服务）

2. 修改PHP程序

在php文件中 加入

注意:为了系统的其它资源的正常使用 请您不要将 memory_limit设置太大，其中-1为不限

3. 修改.htaccess 文档

修改前提是前提是该目录支持.htacces

在文档中新增一句：php_value memory_limit 16M(或更大)
