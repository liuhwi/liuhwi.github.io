限制在Windows 2003 的终端连接数

发表于 2008-01-10 | | 阅读次数

如何限制在终端服务器上的远程会话的数量

  • 若要打开终端服务配置工具，单击 开始，指向 管理工具，然后单击 终端服务配置。
  • 在控制台树中单击 连接。
  • 在右侧窗格中 rdp-tcp，用鼠标右键单击，然后单击 属性。
  • 在 网卡 选项卡上单击从 最多连接 列表中选择 1。
  • 在 权限 选项卡上单击 添加、 输入对象名称来选择 (示例) 框中键入 所有人 单击 检查姓名框中，然后单击 确定。
  • 在 组或用户名称 区域中单击以选中 Everyone 组。
  • 每个用户的权限 区域中单击以选中要拒绝权限的 来宾访问，拒绝 复选框，然后单击 确定。

备注

  • 此设置允许称为会话 0 的只有一个远程连接。可以只通过控制台进行此连接。
  • 连接的基于 Windows XP 的计算机上或在一台基于 Windows Server 2003 的计算机上的终端服务器控制台会话时根据使用以下命令：
  • 若要将该服务器的远程桌面连接使用 mstsc.exe /console 命令。
  • 如果您使用远程桌面连接 6.1 客户端或更高版本的客户端，使用 mstsc.exe /admin 命令。
  • 有关远程桌面连接 6.1 客户端的详细信息，请单击下面的文章编号，以查看 Microsoft 知识库中相应的文章：
